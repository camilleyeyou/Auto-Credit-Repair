**backend/models/letter.py** — Pydantic models (per D-09 and research Pattern 1):
```python
from typing import Optional
from pydantic import BaseModel

class LetterRequest(BaseModel):
    bureau: str                          # "experian" | "equifax" | "transunion"
    creditor_name: str
    account_number_last4: Optional[str] = None
    dispute_reason: str
    fcra_section: str                    # e.g. "611"
    fcra_section_title: str             # e.g. "Right to Dispute"
    fcra_section_usc: str               # e.g. "15 U.S.C. § 1681i"
    # User profile fields (per D-04)
    full_name: str
    street_address: str
    city: str
    state: str
    zip_code: str

class LetterResponse(BaseModel):
    letter_html: str      # full rendered HTML stored in dispute_letters.letterContent
    pdf_base64: str       # base64-encoded PDF bytes uploaded to Convex Storage by action
```

**backend/templates/letter.html** — HTML letter template (per D-02, D-05, D-06, D-07, D-08):

Use an f-string template approach (not Jinja2 — no new dependency). The template is built as a Python function `render_letter_html()` in letter_writer.py that returns a complete HTML string. The HTML structure must be:
- `<style>` block with: `@page { size: 8.5in 11in; margin: 1in; }` and `body { font-family: "Liberation Serif", Georgia, serif; font-size: 12pt; line-height: 1.5; color: #000; }`
- Sender block: date (formatted as "April 3, 2026"), then user full_name, street_address, city/state/zip — each on its own line via `<br>`
- Bureau address block: bureau display name + hardcoded mailing address (D-03)
- RE line in bold: `RE: Dispute of Inaccurate Credit Information — Account Ending in {account_number_last4 or "N/A"}`
- Salutation: `Dear {bureau_display_name} Dispute Center,`
- Body paragraph: Claude-generated text (unique per item)
- Certified mail note: `This letter is being sent via USPS Certified Mail.`
- Signature block (D-06): `Sincerely,` then `<div style="height:3em;"></div>` then user full_name
- Enclosures (D-07): `Enclosures: Copy of government-issued ID, Copy of relevant credit report page`

Bureau address constants (D-03) — hardcode in letter_writer.py:

BUREAU_ADDRESSES = {

   - Returns complete HTML string with all sections described above

3. `def html_to_pdf_bytes(html_string: str) -> bytes:`

**backend/routers/letters.py** — FastAPI router (mirrors reports.py pattern):
```python
router = APIRouter(prefix="/api/letters", tags=["letters"])

@router.get("/health")
def letters_health():
    # Validate WeasyPrint is importable and functional
    from weasyprint import HTML
    HTML(string="<p>test</p>").write_pdf()
    return {"status": "ok", "service": "creditfix-letters"}

@router.post("/generate")
async def generate_letter(request: LetterRequest) -> LetterResponse:
    import base64
    body = await generate_letter_body(request)
    html = render_letter_html(request, body)
    pdf_bytes = html_to_pdf_bytes(html)
    return LetterResponse(
        letter_html=html,
        pdf_base64=base64.b64encode(pdf_bytes).decode("utf-8"),
    )
```
Handle exceptions: if generate_letter_body or html_to_pdf_bytes raises, let FastAPI return 500 — do NOT swallow the error (the Convex action's per-item try/catch handles this gracefully).
