Create `frontend/convex/creditReports.ts` with these exports and internal functions.

**Imports:**
```typescript
import { mutation, query, action, internalMutation, internalQuery } from "./_generated/server";
import { v } from "convex/values";
import { internal } from "./_generated/api";
```

**Public mutation: `generateUploadUrl`**
- Args: none
- Handler: `return await ctx.storage.generateUploadUrl()`
- Requires auth: yes (throw "Not authenticated" if no identity)

**Public mutation: `saveReport`**
- Args: `{ storageId: v.id("_storage"), bureau: v.union(v.literal("experian"), v.literal("equifax"), v.literal("transunion")) }`
- Handler: get identity → insert into `credit_reports` with `userId: identity.subject`, `bureau`, `storageId`, `uploadedAt: Date.now()`, `parseStatus: "uploaded"`
- Returns the inserted document ID

**Internal query: `getReport`**
- Args: `{ reportId: v.id("credit_reports") }`
- Handler: `return await ctx.db.get(args.reportId)` — throw if null ("Report not found")

**Internal mutation: `setParseStatus`**
- Args: `{ reportId: v.id("credit_reports"), status: v.union(v.literal("parsing"), v.literal("done"), v.literal("failed"), v.literal("image_only")), errorMessage: v.optional(v.string()) }`
- Handler: `await ctx.db.patch(args.reportId, { parseStatus: args.status, ...(args.errorMessage ? { errorMessage: args.errorMessage } : {}) })`

**Internal mutation: `saveParsedData`**
- Args: `{ reportId: v.id("credit_reports"), parsedData: v.any(), rawText: v.optional(v.string()), confidence: v.optional(v.number()) }`
- Handler: patch the record with `parseStatus: "done"`, `parsedData`, `rawText`, `confidence`

**Public action: `parseReport`**
- Args: `{ reportId: v.id("credit_reports") }`
- Handler (follow Pattern 1 exactly, with Pitfall 2 null-check and Pitfall 5 env var):
  1. `const report = await ctx.runQuery(internal.creditReports.getReport, { reportId: args.reportId })`
  2. `const fileUrl = await ctx.storage.getUrl(report.storageId)` — if null, call `setParseStatus` with status "failed" and errorMessage "Storage URL not found — file may be missing" then return
  3. `await ctx.runMutation(internal.creditReports.setParseStatus, { reportId: args.reportId, status: "parsing" })`
  4. Wrap in try/catch: `const fastapiUrl = process.env.FASTAPI_URL` — if undefined, throw `new Error("FASTAPI_URL environment variable not set in Convex")`
  5. `const response = await fetch(fastapiUrl + "/api/reports/parse", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ bureau: report.bureau, pdf_url: fileUrl }) })`
  6. If `!response.ok`: call `setParseStatus` with status "failed" and errorMessage `FastAPI error ${response.status}: ${await response.text()}` then return
  7. `const parsed = await response.json()`
  8. If `parsed.parse_status === "image_only"`: call `setParseStatus` with status "image_only" and errorMessage "This PDF appears to be image-only (scanned). Please upload a text-based PDF from annualcreditreport.com." then return
  9. Call `saveParsedData` with `parsedData: parsed`, `rawText: parsed.raw_text ?? undefined`, `confidence: parsed.confidence ?? undefined`
  10. Catch block: call `setParseStatus` with status "failed" and errorMessage from the caught error

**Public query: `listByUser`**
- Args: none
- Handler: get identity → `return await ctx.db.query("credit_reports").withIndex("by_user", q => q.eq("userId", identity.subject)).collect()`
- Returns array of credit_reports documents
